### RESP3.build.verbatimString(str, type)

---

**Type**: method

**Async**: No

**Parameters**:

| Name   | Datatype | Optional | Description |
|--------|:--------:|:--------:|-------------|
| `str`  |  string  |    NO    |             | The string you wish to embed |
| `type` |  string  |    NO    |             | The string you wish to embed |

<br>
**Returns**: `<string>`

---

Returns a string with the RESP3 representation of the passed string as `verbatimString`.

The `type` parameter must be a 3 character string
