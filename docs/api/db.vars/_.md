### _(...subscripts)

Applies to:

- [Globals](../../namespace.db.vars.md)
- [Vars](../../namespace.db.vars.md)

---

**Type**: method
**Async**: no

**Parameters**:

| Name            |    Datatype     | Optional | Description                                                                                     |
|-----------------|:---------------:|:--------:|-------------------------------------------------------------------------------------------------|
| `...subscripts` | string / number |   Yes    | An unlimited number of subscripts, as strings or numbers (or variables) to specify global nodes |

<br>
**Returns**: `glvn`

---

The `_()` (underscore) method allows you to select nodes (subscripts) in a global or a var.

You can have as many subscripts as you want, including none (which is the equivalent to the root of the global / var).

Each subscript can be either a string or a number, and variables can be used.

### EXAMPLES

---

All the following syntaxes are correct.

````js
const res = await mind.db.globals.testGlobal._().decrement()

const res = await mind.db.globals.testGlobal._("firstSubscript", 23, "address").decrement()

const res = await mind.db.globals.testGlobal._(recordId, dptNumber).decrement()

````
